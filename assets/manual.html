PreMoApp User Manual

App Purpose

The PreMoApp is design for manage your day to day memories. User can add memories for particular person(s) too called Nominee. While writing the post user can also add current location and weather. The application is available for Android and IOS users.

The document provides the features and functionality of PreMoApp.

How to Download the App

Download the app from Play store or App Store as per Operating System of your Mobile Phone

User Roles

There are two types of users based on the Role they have:-

A) Normal User and

B) Admin User

A) Normal User: The entire registered users are Normal user. And can use the app with below features.

Application Features

1) Registration Screen

A user must be registered to use the app.

Process:

Click on Create Account Link

Enter User Name

Select country code and then enter Mobile No

Enter Email Id

Enter Password

Confirm Password

Click on Sign Up Button

Accept Terms and Conditions

Choose Pass Code for security or registered without Pass Code

Click on Submit Button.

Login Screen

User must have to Login if already registered.

Process:

Enter Valid Email Id

Enter Password

Click on Sign In Button

Enter your Security Pass Code if enable

Home Screen

User can perform several operations from Home Screen.

1) Add Post (Memory)

Process

Click on Add button (Indicated by 1 in Image)

Click on calendar to select desired date

Select star to Cherish Memory (It just like a favourite memory)

Enter Title for memory

Enter description of Memory

Select Location, Weather, Nominee, and attached Photo (This are optional features)

Click on Post Button to save the post (Memory).

2) Gallery

It shows all the images attached in post.

To see the Images Click on Gallery button

3) Cherished

It shows cherished memories. While adding post if user select star button only these memories will be seen in cherished.

To see the cherished memory Click on the Cherished button.

4) Settings

4a) Change PassCode

User can change or add PassCode for security reasons, Passcode must be 4 digit.

4b) Manage Nominees

Nominees are the person(s), the user may write the memory for him/her, or tag the person(s) in the post (memory). User can create Maximum 3 Nominees

Process:

Click on Add button

Click on + button form right bottom of screen

Click on Name

Select country code and enter Mobile No.

Enter Email Id

Click on Submit Button

4c) Manage Profile

User can personalize his/her profile by adding photo date of birth

Click on Edit Button to manage your profile.

4d) Logout

User can safely logout the session. Click on Logout button to exit from App.

5) Menu Button

Various menu options can explore using this feature. Like Home, Cherished Memories, Gallery, Settings.

6) Search

User can searched saved memory using this feature.

Process:

Click on Search button

Enter search text – It will show relevant memories.

7) Calendar

User can get details about posted Memory. User can toggle between current screen and calendar.

a) Current Date is indicating by Orange colour circular round.

b) Memories are indicating by Blue colour Dot if memories are written for this date.

8) Add Nominees

User can add new nominees. Nominees are not more than 3

9) Help

User can get quick help about how to use the app and FAQ, Can read Terms and Conditions, Privacy Policy.

B) Admin User

Initially all registered user are Normal user, Owner of application may assign the Admin role to user.

To get Admin role you need to contact App owner or company.

Admin can use the entire feature which is discussed in Normal User Role. Additionally Admin can use Manage Quote and App Settings feature.

Settings

1) Manage Quotes

User Can Add New Quote and Edit the existing Quote.

To add the Manage Quote click on “Add” button. Already exist Quote list will be shown. Click on + button to add new Quote. To choose the date click on calendar button, by default calendar show current date. Add Quote in description. Click on Submit button.

2) App Setting

User can set font size, Turn On or Off Daily Quote on Home Page and Splash screen.

If user want to change the font size of Quote displayed on Home screen then user can change the quote size accordingly.

To change the Quote size

Click on Edit button (Top Right corner)

Click on Quote Font Size

Change font size by entering font size

Click on Submit button

3) Change Pass Code

User can Change or Add new PassCode to secure the app by accessing unauthorised person.

To Add / Change Passcode

Click on Settings

Click on Reset button

Enter your desired 4 digit passcode

Click on Submit button

4) Manage Nominee

Nominees are the person(s), the user may write the memory for him/her, or tag the person(s) in the post (memory). User can create Maximum 3 Nominees

Process:

Click on Add button

Click on + button form right bottom of screen

Click on Name

Select country code and enter Mobile No.

Enter Email Id

Click on Submit Button

5) Manage Profile

User can Set or Change the Profile photo. Also can edit Name and Mobile no. User can add Date of Birth and disclosing Date.

Click on Settings

Click on Manage Profile Edit Button

Click on Edit button

Now you can change Name, Mobile No, Add Date of Birth and disclosing date

User also can add or change pic

To change the photo

Click on Camera button

Select the source to add Photo Gallery or Camera

Click on Submit button

6) Logout

Users can safely logout from the App. Click on Logout button to Exit from App.
